### 1. Add Your Content (CRITICAL)

#### Replace Placeholder Images
Current placeholder images need to be replaced with real ones:

- `images/profile.jpg` - Your professional headshot (800x800px)
- `images/og-image.jpg` - Social media preview (1200x630px)
- `images/logos/tudublin.png` - TU Dublin logo

- `images/logos/lautech.png` - LAUTECH logo
- `images/research/*.jpg` - Research project images (4 images)
- `images/blog/*.jpg` - Blog post featured images (3 images)

**Image Requirements:**
- Professional headshot: 800 x 800 pixels, < 200KB
- Featured images: 1200 x 600 pixels, < 200KB
- Logos: ~200px width, PNG with transparency preferred
- Compress all images: use TinyPNG.com or Squoosh.app

#### Add Your CV
- Export your CV as PDF
- Name it exactly: `CV_Azeez_Hamzat.pdf`
- Place in `docs/` folder

#### Configure Contact Form
1. Go to https://formspree.io
2. Sign up (free)
3. Create a new form
4. Copy your form endpoint
5. In `contact.html`, line 123, replace:
   ```html
   <form action="https://formspree.io/f/YOUR_FORM_ID" method="POST">
   ```
   with your actual form ID

### 2. Review and Customize Content

#### Text Content to Review
All the content is based on your information, but review and adjust:
- Homepage about section
- Research page descriptions
- Publication abstracts
- Blog post sample content (or delete and write your own)

#### Customization Options
If you want to change colors:
- Edit `css/variables.css`
- Change `--color-primary` and other color variables

### 3. Test Everything Locally

Before deploying:
- ✅ Open `index.html` in your browser
- ✅ Click through all navigation links
- ✅ Test mobile view (F12 → Device Toolbar)
- ✅ Check all images load
- ✅ Test contact form (after Formspree setup)
- ✅ Read through all text for accuracy

### 4. Deploy Your Website

Choose a hosting platform (see DEPLOYMENT_GUIDE.md):
- **Recommended: Netlify** (easiest, best performance)
- Alternative: GitHub Pages
- Alternative: Vercel

Follow the step-by-step instructions in DEPLOYMENT_GUIDE.md

### 5. Connect Your Domain

Configure DNS for azeezhamzat.com:
- Add A records or CNAME as instructed by your hosting platform
- Wait 24-48 hours for DNS propagation
- Enable HTTPS

## 💡 Important Notes

### Blog System
The blog is designed to be **extremely easy** to update:
1. Write posts in Markdown (simple text formatting)
2. Add one entry to `posts.json`
3. Done!

See BLOG_GUIDE.md for detailed instructions.

### Browser Storage NOT Used
As requested, the site does NOT use localStorage or sessionStorage. All blog posts are loaded fresh from the JSON file each time.

### No Build Process
This is pure HTML/CSS/JS - no build process, no npm, no dependencies. Just open the files and they work!

### Framework-Free
No React, Vue, or Angular. This makes the site:
- Faster to load
- Easier to understand
- Won't break when frameworks update
- Simple to maintain

## 🆘 Troubleshooting

### Images Don't Load
- Check filename matches exactly (case-sensitive!)
- Verify image is in correct folder
- Check file path in HTML

### Blog Posts Don't Appear
- Check `posts.json` for syntax errors
- Use jsonlint.com to validate
- Verify markdown file path is correct

### Contact Form Doesn't Work
- Make sure you replaced YOUR_FORM_ID
- Check Formspree dashboard
- Verify email address in Formspree

### Site Looks Broken
- Open browser console (F12)
- Look for red error messages
- Check if CSS files are loading
